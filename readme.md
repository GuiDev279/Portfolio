                                         Portfólio
Este é um portfólio pessoal desenvolvido para apresentar projetos, habilidades e informações de contato. Ele é responsivo, moderno e otimizado para diferentes dispositivos.

📂 Estrutura do projeto

Portifolio/

├── index.html              # Página principal

├── src/

│   ├── css/

│   │   ├── reset.css       # Reset de estilos

│   │   ├── responsivo.css  # Estilos para dispositivos móveis

│   │   └── style.css       # Estilos principais

│   ├── imagens/            # Imagens utilizadas no site

│   └── js/

│       └── index.js        # Scripts JavaScript


🚀 Tecnologias utilizadas
HTML5 para marcação semântica

CSS3 (incluindo media queries) para estilo e responsividade

JavaScript para interatividade

Design Responsivo para melhor experiência em diferentes telas

✨ Funcionalidades
Apresentação pessoal e foto de perfil

Destaque para projetos com imagens e descrições

Layout adaptável para celulares, tablets e desktops

📜 Licença
Este projeto é de uso pessoal. Você pode utilizá-lo como base para criar seu próprio portfólio, mas personalize o conteúdo e imagens.
